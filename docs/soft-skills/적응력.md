# 적응력

기술적인 변화 말고도 환경적인 변화를 모두 고려하여 적응하는 능력.

정보가 부족한 상황에서 효율적으로 일을 하기 위한 능력.

---

## 사례 1. 도메인 용어집

백업 스케줄, 백업 등 다양한 혼재되는 용어로 하나의 개념을 표현할 수 있다.

가령, 서버라고 할 수 있지만 실제로는 가상화된 서버, 즉 VM이라고 표현할 수도 있다.

이에 대해서 팀에서 같이 사용할 수 있는 도메인 언어, 즉 유비쿼터스 언어집(Glossary)를 정의하여 혼란을 줄이고 일관성을 유지하려고 노력하였다.

---

## 사례 2. 오브젝트 스토리지 BFF 개발

오브젝트 스토리지 장비 및 게이트웨이를 제어하는 API를 개발하는 과정에서 해당 장비가 바로 들여오지 못하는 상황이었다.

이를 해결하기 위해, S3를 호환하는 API 전제 조건을 고려하여 테스트 컨테이너를 `localstack`을 사용하여 테스트 환경을 구축하였다.

이를 통해서 기능을 구현하고 테스트를 할 수 있는 환경을 구성하였다.

마지막에 Endpoint를 정의하여 개발 환경에 바로 발 맞추어 배포할 수 있도록 하였다.
