# 결과 도출

적절한 품질과 기간 내에 결과를 만들어 내는 역량.

- 주도적으로 결과를 만들어 내는 역량.
- 충분한 회의나 분석을 거치지 않고 문제 해결을 시도하는데만 급급한 경우.
- 팀 내 의사소통.
- 문제를 해결하는데 필요한 **정량적인** 데이터를 수집해 본 경험.
  - "모니터링을 통해 이러한 것을 개선해야 한다"를 제안.
  - "모니터링과 부하 테스트를 통해" 이렇게 개선함.

사례를 쌓으려면 데이터가 있어야 하는데, 데이터를 수집하는 프로세스가 없다면...?
