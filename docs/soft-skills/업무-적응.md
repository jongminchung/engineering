# 업무 적응

팀에 잘 적응할 수록 더 빨리 중요한 일을 맡을 수 있음.

- 생소한 키워드를 문맥별로 모아서 물어볼 것.
- 간단한 질문이 아니라면 30분 ~ 1시간 가량 미팅을 잡는게 좋음.
  - 물어보고자 하는 "주제"에 대해서 자세히 설명하고, 관련된 내용을 미리 준비해두는 것이 좋음.
- 처음부터 너무 많은 정보를 파악하려고 하지말고 나눠서 정복하는 전략을 사용함.

긍정적인 피드백을 받도록 노력해야 함.

- 기술적인 다양성과 표용력을 키울 것.
- 맡은 업무에 대한 책임감.
- 결과를 만들어 내려고 최선을 다할 것.
- 장기적으로 또는 단기적으로 개선하고 싶은 것들을 정리해서 공유함.
- 오너십을 가지고 있어야 함.
- 공유는 적극적으로 하며, 다른 팀원과의 소통을 통해 문제 해결에 도움이 되도록 노력함.
