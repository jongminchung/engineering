# Classpath

라이브러리 `.class` 파일이나 `.jar` 파일을 실행하거나 컴파일할 때, JVM이 클래스 파일을 찾을 수 있도록 경로를 설정하는
것입니다.

## 1. 설정 방법

- **명령행 옵션 (-cp, -classpath)**: 가장 권장되는 방식입니다. 실행 시점에만 적용되므로 유연합니다.

	```bash
	java -cp "lib/*:bin" Main
	```

- **환경 변수 (CLASSPATH)**: 시스템 전역에 설정되어 모든 Java 실행에 영향을 줍니다. 관리가 어려워 권장되지 않습니다.

## 2. 주요 특징

- **와일드카드 (*)**: 특정 디렉토리 내의 모든 JAR 파일을 포함할 때 사용합니다. (단, `.class` 파일은 포함되지 않으며
	디렉토리 경로를 직접 지정해야 함)
- **우선순위**: 클래스패스에 나열된 순서대로 클래스를 탐색합니다. 먼저 발견된 클래스가 로드됩니다.
- **기본값**: 클래스패스를 지정하지 않으면 현재 디렉토리(`.`)가 기본값으로 사용됩니다.

## 3. 구성 요소 구분

- **디렉토리**: 패키지 구조의 루트 디렉토리를 지정합니다.
- **JAR 파일**: 압축된 클래스 파일 묶음 경로를 직접 지정합니다.
